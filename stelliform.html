Stelliform

Worked with team of 7 to build an iOS and Android cross-platform mobile game that features fast-pasted action and strategy with clean, playful design.

TypeSchool project | Cornell University
RoleDesigner
ToolsSketch, Illustrator

Spring 2019

Stelliform is an iOS and Android mobile game that my group members and I developed over the course of a semester. As lead UI designer, in addition to brainstorming game concepts with the group, I created wireframes, mockups, all interface assets, and promotional materials. We released the game at the 2019 GDIAC Showcase and it is available for download on the Google Play store.

The interface





My main role in this project was to handle all design and frontend components. I created and refined a series of mockups in Sketch, then implemented in interface using Flask and jQuery.

The interface allows users to search by specific parameters or search parks similar to their favorite hangouts. It then returns the top 10 matches along with photos, reviews, map, and similarity score explaining the match.

Behind the scenes

Data gathering

Our initial data scrape yielded thousands of reviews from Google Maps, APOPS, and NYC Parks as well as the Wikipedia descriptions of each of the parks listed by NYC Planning on their database.

Preprocessing

My groupmates used a python notebook to tokenize the reviews and create a dictionary mapping outdoor features to summary, which is stored in the applicaiton as a CSV. We generate a good types matrix of relevant words that users can use as search terms.

Analysis

Upon entering the relevant search terms and area, the application takes into account the desired region and similarity to query based upon cosine similarity, sentiment analysis, SVD analysis, and edit distance. It returns a list of weighted similarities and shows the user to top 10 most relevant parks.
